Some tips to use git.
my github address:
https://github.com/God-zuole/note_vim.git
git clone + address	克隆到工作区
git pull	更新工作区到最新版本
git add + filename	将file加到暂存区
git commit -m "message"		将暂存区推到本地仓库(commit),message是我们用来简要说明这次提交的语句
git commit -a -m "message"	将所有已跟踪文件中的执行修改或删除操作的文件都提交到本地仓库	
git status	查看git状态
git push	推到远程仓库
